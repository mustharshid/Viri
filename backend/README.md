# Viri Backend (Laravel)

This directory will contain a Laravel PHP application that provides the REST API for the Viri platform.

## Why Laravel?
- Simple PHP framework, zero‑config on Dhiraagu Pleks.
- Uses built‑in migration system – we will start with SQLite for development and can switch to MySQL in production.
- Serves static assets (if needed) but mainly acts as an API server.

## Setup (once Laravel is installed)
```bash
composer install
php artisan migrate
php artisan serve # for local dev only
```

The backend will expose routes for:
- Tenant and merchant management
- Subscription licensing
- Invoicing
- Append‑only audit log
- Secure endpoint for the extension to request a one‑time token (no credentials stored).
